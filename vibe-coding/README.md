## Lab 1 — Deploy to Azure App Service

> Fork → Create a free App Service → Add one secret → Push and go live.

**[→ Start Lab 1](./azure/)**

What you'll build:
- A live URL on `azurewebsites.net`
- A GitHub Actions pipeline that auto-deploys on every `git push`
- Real cloud infrastructure provisioned from scratch

Prerequisites: free [Azure account](https://azure.microsoft.com/free/) · [GitHub account](https://github.com/join) · [VS Code](https://code.visualstudio.com/) + [Git](https://git-scm.com/)
