## What is MCP?

**Model Context Protocol (MCP)** is an open standard that lets AI models (like Claude, GPT-4, and others) connect to external tools, data sources, and APIs in a standardised way.

Instead of writing custom integrations for every AI client, you build one MCP server and any compatible client can use it.

```
Your Tool / API / Database
        ↓
  MCP Server (you build this)
        ↓
  Any AI Client (Claude, Cursor, VS Code Copilot...)
```
